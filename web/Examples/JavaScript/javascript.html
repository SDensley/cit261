JavaScript Object Examples

Simple Example

JavaScript Objects are a way to programatically describe an object. There are many things that can be done with JavaScript Objects. On this page examples will be shown to demonstrate how you can use JavScript Objects.

JavaScript Objects are created in one of two ways: using an Object Literal or the keyword New.

The format for an Object Literal is as follows:
var object = {name:value, name2: value2, ...};

The format for the New keyword is as follows:
var object = new Object();
object.name = value;
object.name2 = value2;
etc.

Advanced JavaScript Objects

JavaScript Objects can also have other JavaScript Objects as properties as well as methods or functions as part of the object. Press the following button to generate and display some JavaScript objects. The contents are displayed via a method called showContents(). These methods were created via the prototype property.

The JavaScript Code for the two classes is as follows:
// a class to contain the details of a board game
function Game(name, manufacturer, description) {
// object initialization
this.name = name;
this.manufacturer = manufacturer;
this.description = description;
}

// prototype method creation
Game.prototype.showContent = function () {
// return the contents of the object
return "name: " + this.name + ", manufacturer: " + this.manufacturer + ", description: " + this.description.showContent();
}

// a class to contain the details of a description
function Description(players, ages, time, summary) {
// object initialize
this.players = players;
this.ages = ages;
this.time = time;
this.summary = summary;
}

// prototype method creation
Description.prototype.showContent = function () {
// return the contents of the object
return "{ players: " + this.players + ", ages: " + this.ages + ", time: " + this.time + ", summary: " + this.summary + "}";
}
