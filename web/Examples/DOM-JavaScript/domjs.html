DOM Manipulation Example

This page is similar to the Local Storge example page and the JSON and AJAX example page. The differences here is that the way the data is populated onto the page. The data is populated using DOM Manipulation instead of adding HTML tags to a string and applying that as the innerHTML of a div tag.

The page loads automatically and populates with the details about the characters from the game "The Red Dragon Inn: Battle for Greyport," the version they come from, their damage dice for each level, and their abilities for each level.
