CSS3 Animations, Transitions, and Transforms Example

This page displays the contents of pandemic.json in a nice user friendly format. When you click on an image, the top section will fade and slide the details of that current character off the screen and the new character onto the screen. When you mouse over an image it will flip into focus. This is very similar to the transitions and animations with JavaScript page only this time all the transitions and animations are triggered via CSS. with the exception of the fading out and changing the table detailed data.

There is also a cool animation of a spinning card next to the table.

Role Details will display here after selecting a role.
