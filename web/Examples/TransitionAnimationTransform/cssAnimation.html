CSS3 Animations, Transitions, and Transforms Example

This page displays the contents of pandemic.json in a nice user friendly format. When you click on an image, the top section will display the details of that character. This is very similar to the transitions and animations with JavaScript page only this time all the transitions and animations are triggered via CSS.
